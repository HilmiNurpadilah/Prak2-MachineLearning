Tentang Aplikasi Prediksi MPG

Deskripsi Model

Aplikasi ini menggunakan model regresi linear sederhana untuk memprediksi efisiensi bahan bakar (MPG - Miles Per Gallon) berdasarkan berat mobil.

Metodologi

  • Dataset: Auto MPG Dataset (392 sampel)
  • Algoritma: Linear Regression dari scikit-learn
  • Variabel Input: Berat mobil (weight) dalam pound
  • Variabel Output: Efisiensi bahan bakar (MPG)
  • Akurasi: R² = 0.6533, RMSE = 4.2064

Rumus Model

MPG = Intercept + (Koefisien × Berat Mobil)
MPG = 47.2005 + (-0.0079 × Berat)

Interpretasi Hasil

MPG < 15
Efisiensi Rendah
15 ≤ MPG < 25
Efisiensi Sedang
MPG ≥ 25
Efisiensi Tinggi
